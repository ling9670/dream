# Dream #07 — 流浪漢昏迷自醒 | Reel Script
**Phase:** 失去
**Estimated duration:** 50–60 seconds
**Emotional arc:** 一個善意的打開——然後失去控制——然後從最深的地方找回那一個字。

---

## ElevenLabs Script

**Panel 1**
雪下了很久，地面是濕的，路燈的光打在雪上是橘黃色的。他靠著車旁邊睡著了，外套是薄的，一點都不夠擋這種天氣。我走過去，推開了那扇車房的門。

**Panel 2**
門開了，裡面比外面暖一點。他沒有動，頭斜靠著車身，臉色蒼白。我站在門邊，看著他。

**Panel 3**
床就在車房裡，白色的，乾凈的，不應該在那裡的。沒有人放進來，它就在那裡，像是一直都有。他站起來了，走過去。

**Panel 4**
他躺在那張床上，閉上眼睛，身體慢慢鬆開。毯子蓋上去的時候，他的手稍微動了一下。我站在床邊，然後一切變了。

**Panel 5**
我的腳站不穩，地板像是往一個方向傾斜。嘴巴張開了，聲音不出來。我知道我還在那裡——只是身體不再聽從我了。

**Panel 6**
有什麼碰了我一下，我想說停，想說等一等，嘴巴動了，什麼都沒有出來。只有我自己知道我還在裡面。

**Panel 7**
我把那口還在的氣攏起來，說了一個字。<break time="0.5s"/>就那一個字。

**Panel 8**
眼睛睜開了。天花板是我熟悉的顏色，光從窗縫裡進來。我躺在那裡，好一會兒都沒有動。

## Production notes
- Panel 7：「就那一個字」——ElevenLabs 錄音時語速最慢，考慮把這兩句分成兩個檔案，Panel 7a 和 7b，中間 CapCut 插入 0.8s 靜音。
- Panel 8 是解脫的回歸：light comes in——CapCut 可以在這個 panel 加一個非常輕微的曝光提亮（brightness +10），與前幾個 panel 的壓抑形成對比。
- 整個夢不要加任何氛圍特效，只讓靜默說話。
