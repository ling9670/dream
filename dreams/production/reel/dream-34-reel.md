# Dream #34 — 豪宅迷宮父親燒木櫃 | Reel Script
**Phase:** 重建
**Estimated duration:** 60–70 seconds
**Emotional arc:** 一個新的大宅，一個無緣無故的火，一夜的保護——然後早晨來了，一碗豆腐花。

---

## ElevenLabs Script

**Panel 1**
是夜裡，燈沒有開，我從一個側門進去。裡面的地板是大理石的，腳步聲在空曠裡走得很遠。窗外的月光進來，把家具的影子拉長在地板上。我往裡走。

**Panel 2**
走廊連著走廊，房間開著房間，每一扇門打開都是另一扇門。我走了很久，辨不清方向，但繼續走。然後我看見那個房間裡有光。

**Panel 3**
光是從裡面的木櫃反出來的，那個櫃子是深色的木頭，每一個細節都是雕出來的，花和葉子的紋路，門縫是齊的，把手是黃銅的，擦得很亮。我湊近去看，聞到木頭的氣味。他出現了。

**Panel 4**
他拿著火，把那個木櫃點了。火從邊角開始，慢慢往上，木頭燒起來的聲音是低沉的，木材的紋路在火裡發光，然後黑掉。我跑出去找滅火器。

**Panel 5**
滅火器在走廊的另一頭，我跑過去，拉開插銷，跑回來，對著還在燒的地方噴。白色的粉末出來，火小了一點，但他還站在那裡，沒有動。我轉過去看著他。

**Panel 6**
我說了很多話，大聲的，說那個櫃子花了多少功夫，說他為什麼要這樣做，說我不明白。他站在那裡，臉上什麼表情都沒有。我說完了，樓上有聲音。

**Panel 7**
我往樓梯看，他站在最頂，手上拿著槍，指著下面。他沒有動，就站在那裡，槍指著，沒有說話，也沒有射。我也沒有動。

**Panel 8**
天亮了，光從窗縫進來。走廊裡有個人，不是父親，他交給我一個東西——是信封，或者是鑰匙，我沒有看清楚。我轉身找父親，他不在了。走廊裡沒有人，只有光。

**Panel 9**
小車停在豪宅門口，老伯坐在後面，豆腐花還是熱的，冒著白霧。我坐下來，要了一碗。白色的豆腐花在碗裡，糖漿沿著邊緣流下去。<break time="0.5s"/>我吃完了，然後離開。

## Production notes
- Panel 3（木櫃）：這是整個夢裡唯一美麗的東西，讓它慢慢被看見。CapCut 可以加一個非常輕微的暖色 vignette。
- Panel 4：「木材的紋路在火裡發光，然後黑掉」——ElevenLabs 這句有一個節奏，「發光」稍微抬，「然後黑掉」落下。
- Panel 9（豆腐花）：這個 panel 是整個夢情緒的落地，不要急。Hold image 2.5s 後 fade to black。
- 父親燒木櫃是無法解釋的破壞——VO 念到「他為什麼要這樣做」時，不要表演困惑，就陳述，讓觀眾自己去感受那個不能理解的空白。
