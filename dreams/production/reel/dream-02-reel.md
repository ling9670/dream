# Dream #02 — 後樓梯殘缺樂器追逐 | Reel Script
**Phase:** 出發
**Estimated duration:** 50–60 seconds
**Emotional arc:** 灰色的後樓梯、一個破損的存在、一場無法擺脫的追隨——她上了車，坐在後座。

---

## ElevenLabs Script

**Panel 1**
那是一條後樓梯，牆壁是舊的灰色水泥，每一格台階都比正常的高一點點。光是涼的。我站在樓梯頂，往下看，看不見底。

**Panel 2**
我跳著台階往下跑，兩格、三格一起跨，鞋底打在水泥上的聲音在樓梯間裡擴大了好幾倍。轉過一個彎，我停住了。她站在那裡。

**Panel 3**
她的手臂在手腕的地方突然就沒了，截止在那裡。腳邊放著幾件樂器，都是破的——弦斷了，鼓面破了個洞。她沒有說話，我���沒有。

**Panel 4**
我從她旁邊過去的時候，幾乎是貼著她的手臂邊緣。她沒有動，我��沒有停。走過去之後，腳步聲不只是我自己的了。

**Panel 5**
再往下幾層，台階旁邊開始出現更多樂器，靠著牆放著，都是壞的。沒有人在附近，只有那些樂器——和越來越近的腳步聲。

**Panel 6**
腳步聲不急，不慌，穩穩地跟著，每一步都踩在我的步伐間隔裡。我沒有回頭，只是跑快了。

**Panel 7**
門是推開的，外面是日光，刺眼的那種。車停在路邊，門沒有上鎖。我拉開車門坐進去，車開動了。我沒有問司機是誰。

**Panel 8**
我轉過頭。<break time="0.5s"/>她坐在後座，就是她。

## Production notes
- Panel 8 是這個夢最重要的瞬間——VO 說完之後 hold image 至少 2s，讓觀眾坐在那個靜默裡。
- 整個夢色調是灰的，CapCut 可以考慮加一層整體去飽和 LUT（-20%），使視覺與敘事吻合。
- Panel 3（無手腕女生）: 不要加任何特效，就讓那個畫面安靜地存在。
- 「她坐在後座，就是她。」—— ElevenLabs 這句語速要非常慢，考慮手動分段上傳以控制停頓。
